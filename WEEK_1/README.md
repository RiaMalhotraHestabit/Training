# Week 1 — Engineering Mindset

**Objective:**  
Build strong engineering fundamentals — system understanding, debugging, performance measurement, and automation.

## Days Overview
- **Day 1:** System reverse engineering, terminal & Node environment
- **Day 2:** CLI app development, concurrency & performance
- **Day 3:** Git mastery (bisect, revert, stash, merge conflicts)
- **Day 4:** HTTP/API forensics (headers, caching, curl, server)
- **Day 5:** Automation, linting, pre-commit hooks & CI basics

## Output
- Scripts, logs, and reports for each day
- Performance benchmarks and analysis
- Git workflows and debugging practices
- Automation pipeline with validation & build artifacts
