# Week 2 — Frontend Fundamentals (HTML, CSS, JavaScript)

## Objective
Build responsive UI layouts and manipulate the DOM using modern JavaScript (ES6+).

---

## Topics Covered

### HTML
- Semantic HTML5 structure  
- Forms, validation, accessibility  
- Media elements (audio, video, canvas)

### CSS
- Selectors, specificity, box model  
- Flexbox & Grid layouts  
- Responsive design (media queries)  
- Animations & transitions  

### JavaScript (ES6+)
- let vs const, arrow functions  
- Arrays & objects (map, filter, reduce)  
- DOM manipulation & event handling  
- Modular JS & reusable functions  

---

## Days Overview
- **Day 1:** Semantic HTML + blog page  
- **Day 2:** CSS layouts (Flexbox + Grid)  
- **Day 3:** JS ES6 + DOM manipulation  
- **Day 4:** Utilities + LocalStorage (Todo App)  
- **Day 5:** Capstone project (E-commerce UI)  

---

- Built responsive UI layouts  
- Created interactive components using JavaScript
